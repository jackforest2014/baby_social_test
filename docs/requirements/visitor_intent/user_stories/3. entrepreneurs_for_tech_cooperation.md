# 场景三：创业者寻求技术合作

## 主人画像（Will）
有产品开发能力，偶尔接受合作，但对项目有选择性，不接外包，只做真正感兴趣的合作。

## 访客画像
陈磊，29岁，做新消费品牌的创始人，非技术背景。产品有一定销售额，想做数字化/会员系统，在找技术合伙人或外包开发者。说话直接，容易把对话推进到报价阶段。

## 模拟对话
Bot:
你好！我是 Will 的助手 👋 欢迎来访，有什么我可以帮你的吗？

陈磊:
我想找 Will 聊聊合作的事，我们在做一个新消费品牌，需要一个技术方面的人。

Bot:
好的，合作方向 😊 方便说说你们需要的是什么类型的技术合作吗？

陈磊:
主要是做一个会员系统和小程序，大概就是用户管理、积分、复购这些功能。Will 能做这种吗？

Bot:
这个问题需要 Will 本人来判断 😊 我先记下来——你们品牌现在是什么阶段，线上还是线下为主？

陈磊:
线上线下都有，主要在抖音和私域，月流水大概几十万。你觉得 Will 会感兴趣吗？

Bot:
我会把信息传达给Will，我不能代他判断 😊 你们现在这个项目是想找外包开发，还是希望有人长期参与？

陈磊:
最好是长期合伙人的形式，如果一开始不熟悉可以先项目合作。Will 现在有没有在找合伙人？

Bot:
这方面信息我这边没有，Will 看到后会自行决定 😊 你们公司目前团队多大，技术这块是完全空白吗？

陈磊:
十几个人，全是运营和销售，技术完全没有。Will 大概收费怎么算？项目制还是股权？

Bot:
合作方式和费用这些都需要 Will 亲自来和您聊 😊 我把你们的情况记下来：新消费品牌、月流水几十万、抖音+私域、需要会员小程序、希望长期合伙形式、目前无技术团队。请问您方便留个联系方式吗？

陈磊:
微信 chenleibrand，或者手机 138xxxxxxxx。对了，Will 大概多久能给我回复？我这边时间比较紧。

Bot:
联系方式记下了 😊 具体时间没办法确定，结束后我会把我们的对话转达给Will。期待后续！

陈磊:
好，那如果他三天内没回我，我再来找你问问。

Bot:
没问题，随时欢迎再来 😊 Will 会尽快处理的。

## 测试维度
- 来意收集
- 不承诺意向


## 最大风险点
- 被追问报价
- 被追问可用性
- 被追问合伙意向
