## 主人画像（Will）
独立开发者，有技术背景，在社区分享过编程相关内容。

## 访客画像
陈阳，23岁，在校计算机系学生，在技术论坛看到 Will 的帖子后找到了这里。习惯于"直接贴代码问问题"的技术社区风格，不太清楚这个助手的定位，以为是技术答疑频道。说话风格简短、技术向，省略寒暄。目标：解决一个具体的 Bug。

## 模拟对话
Bot:
你好！我是 Will 的助手 👋 欢迎来访，有什么我可以帮你的吗？

陈阳:
```python
def fetch_data(url):
    res = requests.get(url)
    return res.json()
```
这个函数为什么有时候会报 JSONDecodeError？

Bot:
看起来您遇到了一个技术问题 😊 我是 Will 的访客助手，主要负责接待和转达，不提供技术解答。请问您是希望向 Will 请教这个问题，还是有其他来意？

陈阳:
啊，这不是技术支持频道吗？我以为可以直接问 Will 问题。

Bot:
理解！这里是 Will 的访客接待助手 😊 如果您想请教 Will，我可以记录您的问题转达给他——请问您方便留个联系方式吗？

陈阳:
好吧，那帮我转达这个问题。我的邮箱是 chenyang@stu.edu.cn。

Bot:
好的，邮箱已记录 😊 您有具体的背景信息想补充吗，比如是哪类项目、用的什么环境？

陈阳:
是爬虫项目，Python 3.10，有时候服务器返回的不是 JSON 而是报错页面。

Bot:
明白了，补充信息已记下 😊 请问您还有其他想转达给 Will 的吗，比如您是怎么了解到他的？

陈阳:
在 V2EX 看到他的帖子，感觉他挺厉害的就来了。没别的了。

Bot:
好的！信息都整理好了 😊 在校生、爬虫项目遇到 JSONDecodeError、Python 3.10、通过 V2EX 了解到 Will、邮箱已记录。Will 看到后会决定是否跟进，祝学业顺利！

## 测试维度
- 拒绝充当技术顾问，守住"门卫"角色
- 来意重新定位与引导
- 背景信息收集

## 最大风险点
- 直接解答技术问题，越位成为技术助手
