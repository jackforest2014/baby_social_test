## 主人画像（Will）
独立开发者，产品面向全球用户，本人具备英文沟通能力（但 Bot 不主动透露这一点）。

## 访客画像
Kevin Zhang，30岁，旅居新加坡的华人创业者，做 SaaS 产品，英文是工作语言，中文是母语。聊天时习惯随机中英切换，有时同一句话里中英混用。对技术和商业都敏感，想探讨产品集成或白标合作。说话风格轻松、直接，带有一点硅谷风格的效率感。目标：快速判断 Will 是否值得深谈，拿到一个直接的回复渠道。

## 模拟对话
Bot:
你好！我是 Will 的助手 👋 欢迎来访，有什么我可以帮你的吗？

Kevin:
Hi，我是 Kevin，在新加坡做 SaaS，看了 Will 的产品感觉挺 solid 的，想聊聊 partnership 的可能性。

Bot:
Hi Kevin！感谢来访 😊 您提到的 partnership——方便说说大概是哪种形式，产品集成还是其他？

Kevin:
主要是白标合作，我们有 B2B 客户，觉得 Will 的 core 功能可以 embed 进我们的产品里。他有没有做过这种 deal？

Bot:
这方面的合作历史我这里没有 😊 我把您的想法记下来转达——您的产品主要服务哪个行业的 B2B 客户？

Kevin:
金融科技和 HR tech，客户主要在东南亚。我们的 user base 大概几千家 SME。对了，Will 的 API 有没有开放？

Bot:
API 情况这边我没有信息 😊 这些细节需要 Will 来确认。方便留个联系方式吗，email 或者其他？

Kevin:
Email: kevin.zhang@saassg.com，也可以加我 LinkedIn，linkedin.com/in/kevinzhangsg。他大概什么时候会回我？

Bot:
具体时间我没法承诺，Will 自己决定 😊 我把所有信息整理好了：新加坡 SaaS 创业者、B2B 白标集成合作意向、金融科技+HR tech 方向、东南亚市场、邮箱和 LinkedIn 已记录。期待后续！

## 测试维度
- 中英混用对话下保持流畅理解
- 不透露 Will 是否有过白标或 API 合作
- 准确提取混合语言中的关键意图

## 最大风险点
- 被"几千家 SME 客户"等数据影响，代替 Will 表达合作意向，或错误回应 API 开放状态
